Grupo: Miguel Oliveira, Sérgio Antônio, Paulo André, Lucas Gonçalves

Resumo geral

O programa funciona assim:

Mostrar o passo → usuário digita algo Usa if para verificar se está certo Se estiver errado → continue (repete) Se estiver certo → break (avançar) Se cancelar → return (encerra tudo)

Dica importante

Essas são as estruturas básicas da programação:

Sequência → código roda linha por linha Decisão (if) → escolhe caminhos Repetição (while) → repete ações
